# Support

## Questions

Open a GitHub Discussion if discussions are enabled. Otherwise, open an issue with the `question` label.

## Bugs

Use the bug report issue template and include:

- Django version
- Python version
- Package version
- Relevant `DEPLOY_PROBES` settings with secrets removed
- Endpoint response and status code

## Security Issues

Do not open a public issue for vulnerabilities. Follow `SECURITY.md`.

## Supported Versions

Security fixes target the latest released minor version.
